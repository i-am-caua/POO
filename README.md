# Tarefas de Programação Orientada a Objetos (POO) ☕

Bem-vindo ao meu repositório de tarefas da disciplina de **Programação Orientada a Objetos (POO)** do curso de **Ciência da Computação (CC)**.

Este repositório contém os exercícios, projetos e códigos desenvolvidos ao longo da disciplina, implementados na linguagem **Java**.

## 📁 Estrutura do Repositório

O repositório está organizado de forma sequencial, separado por tópicos de aprendizado:

- **`01-Condicionais/`**: Exercícios focados em estruturas de controle e decisões.
- **`02-LacosDeRepeticao/`**: Exercícios com estruturas de repetição (for, while).
- **`03-ExerciciosDiversos/`**: Algoritmos sequenciais e exercícios mistos.
- **`04-Metodos/`**: Criação, chamada e estruturação de métodos.
- **`05-ClassesEObjetos/`**: Introdução à criação de classes e instanciação de objetos.
- **`06-Construtores/`**: Encapsulamento, definição de atributos, métodos e construtores.

## 🚀 Tecnologias Utilizadas

- **Linguagem**: Java
- **Paradigma**: Orientado a Objetos (POO)

## 📖 Sobre

Estes códigos foram desenvolvidos como parte da grade curricular de **Ciência da Computação**, com o objetivo de fixar os conceitos fundamentais de desenvolvimento de software em Java.
